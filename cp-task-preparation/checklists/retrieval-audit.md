# Retrieval Audit Checklist

- `SKILL.md` routes to the new or changed guidance.
- The guidance has a clear trigger for when to load it.
- The guidance lives in one canonical owner.
- Related checklists point to the owner instead of repeating methodology.
- No old filename or obsolete route remains.
- A future session can find the rule without loading unrelated references.
- Key duplicated concepts were checked with `rg`; stale-limit/routing searches exclude generated tests and answers unless generated data is the audit target.
